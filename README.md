🌴 TodoList
Jwt를 사용하여 인증 / 인가 기능 구현한 TodoList 기능<br>
<br>

💻 프로젝트 설명
기술과 개발의 다양한 분야를 '기술 나무'에 비유하여, 개발자들이 자신의 경로를 찾아가는 데 도움을 준다는 의미를 담고 있습니다.<br>
<br>

✨기능
🧑User : 로그인, 로그아웃
🧑Todos : 할일 목록 작성, 수정, 삭제
🧑Comments : 댓글 작성, 수정, 삭제<br>
<br>

⏱ 개발 기간
2024.01.16 ~ 2024.01.19<br>
<br>
